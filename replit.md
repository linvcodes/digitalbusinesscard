# Kristina Mladenova Digital Business Card

A cyberpunk-themed, mobile-first digital business card for an Interactive Ads Expert.

## Features
- Terminal-style boot animation
- Neon cyberpunk aesthetic
- Responsive layout (mobile & desktop)
- LinkedIn, GitHub, and Portfolio links
- ASCII art watermark

## Tech Stack
- Vanilla HTML
- CSS (with neon effects and scanlines)
- Vanilla JavaScript (for animations)

## Setup
- Served via `npx serve` on port 5000.
- Profile image: `profile.jpg` (copied from `attached_assets/`).
